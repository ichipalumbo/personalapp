# 💪 Agenda Personal Trainer

Sistema de agendamento de aulas para personal trainers.

## 🚀 Funcionalidades

- ✅ Cadastro de alunos (nome, telefone, preço, objetivo)
- ✅ Agenda semanal (Seg-Sex, 6h-20h)
- ✅ Calendário mensal com visualização de ocupação
- ✅ Agendamento com horário duplo (início/fim)
- ✅ Cores por objetivo (Funcional, Hipertrofia, etc.)
- ✅ Cancelamento de aulas
- ✅ Exportar/Importar dados (JSON)
- ✅ Persistência no navegador (localStorage)
- ✅ Design responsivo e escuro/esportivo
